<span data-ttu-id="bb012-107">Surface Dock 2 は、IT 管理の簡素化を目的として設計されています。管理者は、Windows Update を使用して、または社内のソフトウェア配布ツールを使用して更新プログラムを一元管理できます。</span><span class="sxs-lookup"><span data-stu-id="bb012-107">Surface Dock 2 is designed to simplify IT management, enabling admins to automate firmware updates using Windows Update or centralize updates with internal software distribution tools.</span></span>
