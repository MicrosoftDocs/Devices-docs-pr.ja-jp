# Surface 上の Windows 仮想デスクトップ

## はじめに

Surface デバイス上の Windows 仮想デスクトップでは、ローカルデスクトップエクスペリエンスと、タッチ、ペン、インク、および生体認証が物理と仮想の両方の環境の間にある仮想デスクトップの間の境界線で、仮想デスクトップインフラストラクチャ (VDI) を実行できます。 コンピューティングの進化における別のマイルストーンの表現: Surface 1 上の Windows 仮想デスクトップで <a href="#1"><sup> </sup></a> は、高度なセキュリティ保護、エンタープライズレベルの管理性、および Windows 10 Surface on surface の強化された生産性ツールを使用して、Microsoft 365-仮想化された Azure cloud が統合されています。  Azure の premium フォームファクターと仮想デスクトップインフラストラクチャのフュージョンは、ユーザーエクスペリエンス、移植性、セキュリティ、ビジネス継続性、および先進の管理において、例外的な顧客価値を提供します。

### Windows Virtual Desktop

Windows 仮想デスクトップ (WVD) は、Azure クラウドで実行される包括的なデスクトップとアプリの仮想化サービスです。 これは、簡素化された管理、マルチセッション Windows 10、Office 365 ProPlus の最適化、リモートデスクトップサービス (RDS) 環境のサポートを提供する唯一の仮想デスクトップインフラストラクチャです。 WVD を使用すると、Windows デスクトップとアプリを Azure に簡単に展開してスケーリングし、セキュリティとコンプライアンスの機能を組み込むことができます。

### Windows 仮想デスクトップパートナーの統合

Windows 仮想デスクトップ用の認定パートナープロバイダーと独立系ソフトウェアベンダーの一覧については、「 [Windows 仮想デスクトップパートナーの統合](https://docs.microsoft.com/azure/virtual-desktop/partners)」をご覧ください。 一部のパートナーは、サービスとして仮想デスクトップを提供する (DaaS) こともあります。 DaaS は、完全に管理された、ターンキーのデスクトップおよび仮想化サービスを提供することによって、仮想マシン (Vm) を保持する必要がなくなります。 カスタマイズされたデスクトップを世界中のユーザーに配信する機能を利用することで、企業はオンデマンドでクラウドデスクトップのオンデマンドをスピンアップすることができます。

### 仮想 Gpu

Gpu は、AI による計算とグラフィックスを多用するワークロードに適しており、ハイエンドのリモートビジュアル化、ディープラーニング、予測分析などのシナリオを通じて、顧客が革新を生み出していくのに役立ちます。  ただし、内部の GPU の馬力の変動は物理デバイスに関連しており、機動性と柔軟性が制限されているため、リモートまたは外出先での作業が必要な場合は、この方法は適していません。
 
この問題を解決するには、NVIDIA の GPU 機能 (vGPU) を使用した N シリーズの仮想マシンが提供されます。 VGPUs では、複数の仮想マシン間で GPU のパフォーマンスを共有することも、複数の Gpu を1つの仮想マシンに割り当てることによって電力を要求することもできます。 Surface の場合は、使用しているデバイスに関係なく、非常にポータブルな surface からスリムでスタイリッシュな Surface Pc 3 に接続して、デバイスから強力なサーバークラスのグラフィックスパフォーマンスにアクセスできることを意味します。  Surface と vGPUs を使用すると、ペン、タッチ、キーボード、トラックパッド、ピクセル表示の画面を含め、高パフォーマンスのコンピューティング環境でのみ使用可能なグラフィックス機能を備えた、便利な機能をすべて組み合わせることができます。 
 
Azure N シリーズでは、これらの機能を Surface デバイスで利用できるようになるため、どこからでも自由に作業を行うことができます。  [Azure N シリーズと GPU で最適化された仮想マシンのサイズについて説明します。](https://docs.microsoft.com/azure/virtual-machines/sizes-gpu)

### セキュリティと管理

チップからクラウドに向けて、Microsoft 365 および Surface は、組織が保護された状態を維持するのに役立ちます。
Microsoft によって設計、構築、テストされた Surface ハードウェアとソフトウェアの両方を使用すると、ユーザーは、チップからクラウドまでの主要技術によって、生産的で保護されていることを確信することができます。  リモートで作業しているユーザーの数が増加すると、企業データと知的財産の保護がこれまでよりもさらに重要になります。 Surface 上の Windows 仮想デスクトップは、すべてのアクセス要求が厳密に認証され、ポリシーの制約内で承認され、アクセス権を付与する前に異常を検査する、ゼロ信頼のセキュリティモデルに基づいて設計されています。
 
先進の管理により、クラウドコンピューティングの効率性が最大限に向上し、競争力のあるビジネス環境でのユーザー、関係者、お客様のニーズに応えることができます。 たとえば、チームの操作を最小限に抑えて、Surface デバイスをすばやく実行できます。 セットアップは自動で、セルフサービスが提供されます。 チームとユーザーの両方について、更新プログラムが迅速かつ簡単に更新されます。 デバイスの物理的な場所に関係なく、デバイスを管理することができます。
 
Windows 仮想デスクトップでは、次のようなセキュリティと管理機能が提供されます。

- **Windows Update。** Windows を最新の状態に保つことにより、新しいセキュリティの脅威を事前に把握することができます。 Windows 10 は、セキュリティを強化するために、地上から強化されています。また、最新のハードウェア機能を利用してセキュリティを強化します。 <a href="#5"><sup>ビジネス向けの UEFI 5 および Windows Update を使用して、 </sup></a> 進化する脅威に対応しているので、エンドツーエンドの保護はセキュリティで保護され、簡素化されています。

- **リムーバブルドライブ**-新しい Surface デバイスのサブセットは、取り外し可能な SSD ドライブ6を使って、 <a href="#6"><sup> </sup></a> データの保持をより細かく制御できる機能です。
- **先進認証-** Microsoft 365 と Surface は、すべての Windows セキュリティ機能 (ライセンスと有効化の対象) を実現するための統合されたプラットフォームです。 すべての Surface ポートフォリオデバイスは、Windows Hello for Business 向けに設計された、デバイス上から VDI ベースのエクスペリエンスにシームレスに対応する生体認証セキュリティを提供します。
- **最新のファームウェア管理**-デバイスファームウェア構成インターフェイス (dfci) を使用して、 <a href="#7"><sup> 7 </sup></a> IT 管理者は、マイク、USB ポート、SD カードスロット、カメラ、Bluetooth などのファームウェアレベルで、ハードウェア要素をリモートで無効にすることができます。 Windows Defender Credential Guard は仮想化ベースのセキュリティを使用して、特権を持つシステムソフトウェアだけがアクセスできるようにします。
- **下位互換性と上位互換性**-Windows 10 デバイスでは、ハードウェア、ソフトウェア、サービス間の下位互換性と上位互換性が提供されます。 Microsoft は、最新のテクノロジを取り入れながら、ハードウェア、周辺機器、ソフトウェア、サービスの従来のサポートを維持するための強力な履歴を持っています。 企業は、長期の耐用年数を備えた投資を計画することができます。
- **従来の windows 7 ワークロードのブリッジ**-従来の windows OS 環境に依存するソリューションシナリオでは、企業は Azure で実行されている windows 7 の VDI インスタンスを使うことができます。 これにより、Surface などの最新のデバイスで、最新のセキュリティ更新プログラムを受信できなくなる以前の Windows 7 コンピューターに依存するリスクがなくなります。  以前のバージョンの Windows 10 ハードウェアが既に展開されている場合、これらの "将来の校正" 特典に加えて、従来のワークロードの移行が大幅に簡素化されます。
- **ゼロタッチ展開**-自動操縦は Surface デバイス向けの先進管理展開オプションとして推奨されます。 Surface on Surface は windows 10 のクラウドベースの展開テクノロジです。 Surface で Windows 自動操縦を使って、ゼロタッチプロセスでデバイスをリモートで展開して構成することができます。 Windows 自動操縦装置-登録済みデバイスは、最初の起動時に、ハードウェアハッシュと呼ばれる一意のデバイス署名を介してインターネット経由で識別されます。 これらのユーザーは、Azure Active Directory (Azure AD) やモバイルデバイス管理などの先進の管理ソリューションを使用して、自動的に登録および構成されています。

### Surface デバイス: 環境への影響を最小限に抑える

Surface は、ライフサイクル評価を実行して、製品のライフサイクルの主要なステージ間でデバイスの環境への影響を計算します。これにより、Microsoft がこれらの影響を最小限に抑えます。 各 Surface 製品には、温室性ガスの排出量、主要エネルギー消費量、およびマテリアル構成データ、梱包、リサイクル、関連する条件についての詳細が含まれているエコプロファイルがあります。 各 Surface デバイスのプロファイルをダウンロードするには、「Microsoft ダウンロードセンターの[エコプロファイル](https://www.microsoft.com/download/details.aspx?id=55974)」を参照してください。

## まとめ

Surface 上の Windows 仮想デスクトップでは、ユーザー、関係者、顧客のさまざまなニーズに合わせて、より柔軟性の高い柔軟性を備えた組織が提供されます。 Surface デバイスで Windows 仮想デスクトップソリューションを実行すると、従来のデバイスに対する引き続き依存した機能を利用できます。  Surface Go 2 や Surface Pro 7 など、クラウド (またはオフライン) に接続された柔軟なフォームファクターにより、ユーザーはいつでもどこからでも生産性を高めることができます。 従業員が永続的なオンデマンド、またはジャストインタイムのシナリオで作業するかどうかにかかわらず、一般正常性緊急攻撃またはその他の予期しないイベントによって中断されている生産性を維持するための汎用性を備えた企業が、Windows 仮想デスクトップを利用することができます。 組み込みの多層セキュリティと Windows 10 の先進の管理機能を使用すると、企業はクラウドベースのサービスの拡張されたエコシステムを活用して、Windows デスクトップとアプリを迅速に展開し、拡張することができます。 簡単に言えば、Surface 上の Windows 仮想デスクトップは、あらゆる規模の組織や企業にとって非常に必要なテクノロジを提供します。
