# Surface Hub 2S 85 のインストールとマウント"

この記事では、商用環境に Microsoft Surface Hub 2S 85 を物理的にインストールするためのサポート ガイダンスを提供します。 

## ビデオのボックス化を解除する

Surface Hub 2S 85" をインストールする方法に関するビデオが提供されました。 始める前に [、Surface Hub 2S 85](https://aka.ms/Hub2S85Unboxing)のボックス化の解除とインストールを確認してください。

## すべての安全上の予防措置に従う 

> [!WARNING]
> **処理とサイトの準備**<br>
>- デバイスが非常に重い。 デバイスのサイズと重みにより、個人のけが、人の損害、またはデバイスの損傷のリスクを軽減するには、デバイスを直立状態に保つ必要があります。
>- デバイスをインストールする場所に移動する前に、サイトを調査して、展開してマウントする場所に安全に移動する方法を決定します。
>- 開梱とインストールには、常に少なくとも 2 人のユーザーを使用します。
>- デバイスを解凍したら、すぐにマウントする必要があります。したがって、展開を開始する前にマウント システムを配置する必要があります。 ローリング スタンドにマウントする場合は、開梱する前に、スタンドのハンドルをロックまたはブロックします。
>- 危険がつながらな状態を回避するには、アセンブリ領域にパッキング素材を含めしないようにします。

> [!IMPORTANT]
>先に進む前に、以下の付録に記載されている安全に関する追加情報 [を確認](#appendix) してください。

## 1. 外側のパッケージ素材を削除する

1. 外側のカバーを取り外してリサイクルします。
2. 4 本のストラップをカットします。
3. 端から 6 個のクリップを開いて削除します。
4. カバーを取り外し、前面パネルと背面パネルを持ち上げ、取り外します。

 ![外側のパッケージを削除する](images/hub85mount1a.png) <br>

## 2. 黒の外側のパッケージパッケージを削除する

1. 黒い角の部分を削除します (4)。
2. 黒いセンター センターのサポートを削除します。

 ![黒の外側のパッケージパッケージを削除する](images/hub85mount2a.png) <br>

> [!Caution]
> ハブ 2S が配置されるカートまたは壁のマウントの隣に置かれるまで、白い壁、ひな型、またはパッケージ化された素材を取り外す必要があります。
パッケージの背面にある手のひな型の下にある小さなボックスに、追加のひなが付く素材が用意されています。 移動する前、特にデバイスとその保護パッケージを回転する前に、オリジナルまたは交換用のひずみ付き素材を配置する必要があります。
 

## 3. パレットから内側のパッケージ フレームを削除する

1. パレット アセンブリを位置に移動します。
2. ロック ホイールのブレーキ (4)。
3. パレットから内側のパッケージをスライドします。

 ![パレットから内部パッケージ フレームを削除する](images/hub85mount3b.png) <br>

## 4. デザインに合わせてパッケージ フレームを回転させる

1. フレームを回転させて、枠に合わせて調整する
2. 内側のフレームの面取り端で回転し、その内側にスライドします。

 ![パッケージ フレームを回転させてサイズに合わせて調整する](images/hub85mount4a.png) <br>

> [!NOTE]
> ホイール付きパレットは、内部フレームのパッケージ化フットプリントに合わせてカスタムに適合し、インストール サイトの配信プロセス全体で使用できます。 内側の木枠の端には、滑ったプレートがあります。
 
## 5. 施設から削除する

1. スライドアウト
2. ロック ホイールのブレーキ (x4)。

 ![取り外し](images/hub85mount5a.png) <br>


## 6. Surface Hub 85" をパレットに戻す

1. 面取りを使用して、パレットの上に回転します。
2. ホイール ハンドルのロック解除 (x4)。
3. 製品を壁またはカートのマウントの場所に移動します。

 ![Surface Hub 85" をパレットに戻す](images/hub85mount6a.png) <br>

## 7. Surface Hub 85" をウォール マウントまたはカートに配置する

1. ハブ 2S をウォール マウントまたはカートの前に配置します。
2. ロック ホイールのブレーキ (x4)。
3. パレットから内側のパッケージをスライドします。
4. 3 つのストラップをカットします。
5. カバーを取り外します。
6. 白い白い部分を削除します。
7. ウェルカム キットを削除します。
8. 垂直に持ち上げて、端の部分を取り外します。
9. 4 本の手で、下地に木の端の部分を取り除く。

   ![Surface Hub 85" をウォール マウントまたはカートに配置する](images/hub85mount7a.png) <br>

   > [!CAUTION]
   >  Hub 2S 85" を無人のままにしないでください。 デバイスを直立に保持するには、追加のユーザーが必要です。 エンド 部分を取り外した後、最低 1 人のユーザーが、壁のマウントまたはカートへの配置が完了するまでハブ 2S との接触を維持する必要があります。

10. 端を持ち上げると、端が上がって戻ります。
11. ラベルを後ろから取り外します。
12. Hub 2S を下のトレイから離し、カートまたはウォール マウントの上に置きます。

    > [!NOTE]
    > カートまたはウォール マウントの指示に従って、内側のパッケージ トレイからハブ 2S を取り外す前に、マウント システムを準備します。
 
    > [!NOTE]
    > ハブ 2S をつかむ際は、手のホールドが下の段に切り取った点に注意してください。 スピーカーが位置するデバイスを把握しない場合は、上手に注意する必要があります。 デバイス カバーの端にあるグラフィックは、一般的なスピーカーの場所を提供します。

13. 柔軟なクリップ (x2) を緩くします。
14. カバーを取り外します。
15. ペン (x2)、カメラ、電源コードを配置する場所をメモします。
16. ペン (x2)、カメラ、電源コードを接続します。
17. Cling ラベル (x4) を削除します。
18. 右下の電源ボタンを押します。 これでインストールが完了しました。

## 付録 

### その他の安全性に関する情報

> [!WARNING]
> **オブジェクトの重い持ち上げ/作業**<br>
> デバイスが非常に重い。 デバイスに対するリスクを軽減するために、少なくとも 2 名以上のユーザーがデバイスを持ち上げる方法をお勧めします。 デバイスを持ち上げたり、動かしたりするときに、適切な持ち上げの位置を使用することが重要です。
> 以下を含む(ただしこれらに限定されない)、優れた取り上げプラクティスを使用します。
> - 計画を進んでください。 持ち上げチームが計画に同意する必要があります。
>- 単位を持ち上げるかどうかを決定します。 重すぎるか、またはぎこちなさすぎるか。
>- 支援を受け取る必要がある場合に決定します。
>- 環境に障害物や滑り面が見えなされていないか確認します。
>- 後ろではなく足で持ち上げる。
>- 後ろを真っ直ぐに保ち、足を伸らします。
>- 本体の近くに単位を保持します。
>- 本体の中央をユニットの上に合けます。 肩の幅に関するフィートは引き離します。
>- 上に滑らかに上に持ち上げる。
>- Torso を真っ直ぐに保つ。持ち上げたり、負荷が上がった後にひねったりしない


> [!WARNING]
>  **適切なマウント**<br> デバイスが重く、カートまたは壁のマウントに取り付ける。 デバイスに対するけが、人の損害、または損害のリスクを軽減するには、次の方法を使用します。 
>- カートまたはウォール マウントの製造元が提供する指示に従います。
>- 提案されたマウント システムが、このデバイスの重みをサポートします。
>- システムのマウントに備え、取り付けハードウェアのみを使用してください。
>- 製造元の指示に従って、すべての部品が安全に締め付けられるか確認します。
>- 特定のデバイスがシステムの添付ファイル ポイントのマウントに完全に関与するまで、デバイスを解放しない。
>- Microsoft では、お使いのデバイスで使用するために設計されたカートまたは壁の取り付けシステムを使用するようにお勧めします。 Microsoft は、他のマウント システムの使用による損害、損害、または損害に対して一切の責任を負いません。

> [!WARNING]
>  **壁や他のマウント サーフェスの見えのないハザード** <br>
壁や他の取り付け面には、電気ワイヤー、穴線、その他の見えのないハザードや障害物が含まれている場合があります。 目に見えのないハザードを切り取り、ドリルすると、重大な個人のけがやけがをする可能性があります。
インストールの前に、見えのないハザードを特定し、インストール中にこれらのハザードを回避する責任は、インストーラーが負います。 ドリルや切り取りを行う前に、マウント環境を評価し、壁や他の取り付け面に表示されていないハザードが必ず発生しないか確認してください。

> [!WARNING]
> **ヒントハザード**<br>
> 移動時に個人のけが、人の損害、カート/マウントされたデバイスへの損害を回避するには、次の方法を使用します。
> - このデバイスと互換性のあるカート/スタンドのみを使用します。
>- カート/スタンドの製造元が提供する、スタンドアロン デバイスの移動または再配置に関する指示に従います。
>-  デバイスやカート/スタンドで、ハングしたり、重いオブジェクトを配置したりしません。
>-  カート/マウントされたデバイスを移動する前に、必要に応じて電源コードと他のケーブルを取り外します。 カート/マウントされたデバイスを移動する場合は注意が必要で、移動が遅くなります。 カート/スタンドの製造元の指示に従って、スタンドを移動または再配置します。
>-  カート/スタンド マウントデバイスを上下の傾斜台に移動する場合は注意が必要です。 カート/スタンド マウントデバイスは、傾斜台の上または近くに置かれたままにしないでください。
>-  カート/マウントされたデバイスを移動する必要があるのは成人のみです。

> [!CAUTION]
> **タッチ スクリーン のガラス**<br>
デバイス上のタッチ スクリーンは、ほとんどのタッチ スクリーンと同様に、1 つの画面で構成されています。 デバイスが取り下げられたり、重大な影響を受け取った場合、このガラスは壊れる可能性があります。 個人のけがのリスクを軽減するために、画面が壊れている場合は画面にタッチしないようにし、画面を交換する必要があります。
デバイスの誤用または不正使用によって引き起こされたタッチ スクリーンの破損または破損は、製品の限定的な保証の対象ではありません。

> [!WARNING]
>  **適切なインストール**<br>
不適切なデバイスのインストールに関連するハザードを回避するには、作業を開始する前にインストール手順を読んで理解しているユーザーがインストールを実行する必要があります。 必要な機器や専門知識を持たしていない場合、またはマウント面が適切にサポートされているのが不明な場合は、専門のインストーラーに問い合わせください。
